AssignToLayer v1.5.1
Cinema 4D Command Plugin

Oficialni Maxon Plugin ID: 1068663

Instalace:
1) Ukoncete Cinema 4D.
2) Ze slozky plugins odstrante stare verze:
   - AssignToLayer_CS
   - AssignToLayer_EN
   - starsi AssignToLayer
3) Zkopirujte slozku AssignToLayer do slozky plugins.
4) Spustte Cinema 4D.
5) V Command Manageru hledejte AssignToLayer.

Zmeny ve v1.5.1:
- Opravena sirka sloupce v seznamu vrstev.
- Odstraneno horizontalni soupatko pod seznamem vrstev.
- Zachovano automaticke prepinani jazyka podle nastaveni Cinema 4D.
- Zachovano oficialni Plugin ID 1068663.
- Zachovano pamatovani posledni vrstvy a volby prepsat/neprepsat.

Poznamka:
Pokud po vymene pluginu zustava v C4D stara verze, zkontrolujte, ze ve slozce plugins neni duplicitni stara slozka se stejnym pluginem.
